# thor-snippets

This is a vscode snippets plugin for [**Thor**](https://github.com/youngbeen/thor).

## Features

Common used Thor snippets, includes:

* Page Object
* Filters
* Actions
* Fields
* Table Fields
* Options(Select & Cascader)
